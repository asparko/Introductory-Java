A Java Programming Warm-up


This assignment is intended to warm up your Java programming skills. It does not exercise any object-oriented programming concepts. You can (and should) simply use static public class functions. It will, however, exercise basic programming constructs such as loops and conditional execution and require you to use strings and arrays.


The assignment is in three parts that are in increasing level of difficulty called A1Novice, A1Adept, and A1Jedi. The basic idea for all of the parts is to write a program that processes a set of "DNA strands" represented as strings of the capital letters 'A', 'C', 'G', and 'T'. In all three parts, I will provide a class with a main method that creates an input Scanner object and then calls the method process(), providing the Scanner object as a parameter.


You are responsible, in each case, for implementing the process method as described below and produce output using System.out.println(). Please match the format of the example output provided below and do not produce any additional output. You are free to (and in fact encouraged to) create additional class methods to use as helper functions as you see fit.


The set up directions below assume you are using Eclipse. You are not required to but you do need to use the same package and class names. Remember, capitalization counts.


Scanner


This assignment requires you to make use of a Scanner object. You can read the documentation for the Scanner object here. All you should really need is to use the method next() to read the next word from the input. You may also find the method nextInt() to read the next word as an integer useful for the A1Jedi part.


Set Up


To set up the assignment, create a new project in Eclipse called "Assignment1".


In the project, create a package called "a1".


In the package, create three classes called, A1Novice, A1Adept, and A1Jedi.


Add the following line to the top of each class file after the package declaration:

import java.util.Scanner


Edit the contents of each class to look like this:

    public static void main(String[] args) {
        Scanner s = new Scanner(System.in);
        
        process(s);
    }
    
    public static void process(Scanner s) {
    }

A1Novice


This program should process the input as a sequence of white space separated words. Each word (except the last one) represents a strand of DNA and will be comprised entirely of the characters 'A', 'C', 'G', and 'T' representing the four possible nucleobases. The end of the input is signaled by the word "end".

 


The program should count the total number of each nucleobase in all of the strands and then print the results to the output. For example, if the input to the program was:

 

A 
CC 
AATA 
GGG
TTT
end


The program should produce the following output:

A count: 4
C count: 2
G count: 3
T count: 4


In producing the output, please be sure to match the above, including the order of the count report (i.e., first A, then C, etc.).


A1Adept


This program should process the input as A1Novice does, but in addition to producing the counts, it should also keep track of the DNA strand with the smallest and largest number of each of the nucleobases and print those strands to the output. So, given the following input:

A 
CC 
AATA 
GGG
TTT
end


The program might produce the following valid output (there are other outputs that are equally valid because of ties in the counts between different strings):

A count: 4
C count: 2
G count: 3
T count: 4
Low A count: CC
High A count: AATA
Low C count: A
High C count: CC
Low G count: A
High G count: GGG
Low T count: A
High T count: TTT


You can assume that when we test your program, all of the nucleobases will show up at least once.


A1Jedi


This program counts the occurrence of specified patterns in the input strands and produces a report for each pattern. The first input to this program will be an integer indicating how many patterns will be provided (call this N). The next N words to this program will be patterns of nucleobases to search for. Following the patterns will be a sequence of strands terminated by the word "end" as before. For example, given the following input:

4
A
AC
GTA
TA
AABCGTA
ATAGTAC
CCCGGTA
end


The output should be:

Pattern: A Count: 7
Pattern: AC Count: 1
Pattern: GTA Count: 3
Pattern: TA Count: 4


The output report should list the patterns in the same order as they were input.
