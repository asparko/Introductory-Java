Novice


In this part of the assignment, you will write code to implement a
single class called DNAStrandNovice according to the specification
below.  


The name of the class should be DNAStrandNovice.


You should have a single constructor with the following method signature:

 

public DNAStrandNovice(String strand)


The string passed to the constructor should be composed entirely of
'A', 'C', 'G', or 'T' characters. The constructor should check to make
sure that the strand only contains these characters. If any other
characters are included (including lower case a, c, t, or g), the
constructor should terminate with a
runtime exception. The following line of code is the appropriate way to do this:

throw new RuntimeException("Illegal DNA strand");


The DNAStrandNovice object that is initialized by your constructor
should encapsulate the strand string. DNAStrandNovice
should implement the following instance methods:

 

  • public int getACount()
  • public int getCCount()
  • public int getGCount()
  • public int getTCount()
  • public String getStrandString()
  • public int getLength()

 

 


The first four methods return the number of 'A's, 'C's, 'G's, and 'T's
in the strand respectively. The method getStrandString() should return
the string that was used to create the object (i.e., the string that
was originally passed to the constructor). The method getLength()
should return the length of the strand.


Adept

 


Create a copy of your DNAStrandNovice class, renaming it DNAStrandAdept.

 


Modify your DNAStrandAdept class to include the following methods:

 

  • public int getCount(String pattern)


This method should return the number of times that the pattern appears
in the strand.

 

  • public int findPattern(String pattern, int startIndex)


This method returns the position of the first occurence of pattern
in the strand starting at position startIndex. In other words,
findPattern should start looking for the pattern at the position
startIndex where 0 is the index of the first nucleobase in the strand
and getLength()-1 is the index of the last nucleobase in the
strand. If the pattern is not found, then this method should return
-1. If startIndex is less than 0, start looking at position 0. If
startIndex is equal to or greater than the length of the strand, this
method should return -1.

 


Jedi

 


Create a copy of your DNAStrandAdept class, renaming it DNAStrandJedi

 


Modify your DNAStrandJedi class to include the following methods.

 

  • public DNAStrandJedi snip(String startPattern, String endPattern)


This method should first attempt to find the first occurence of startPattern
in the strand. If found, then it should try to find the first occurence of
endPattern that occurs after the match with startPattern. If found, it
should return a new DNAStrandJedi object that represents the substrand
of nucleobases that occurs between the two matches. In other words if
the strand is:


AACATGTACTACTGGTG


and the snip method is called like this:


snip("CA", "A")


then the resulting new DNAStrandJedi object should represent the
strand "TGT" which is found after the first match of "CA" at index 2
and before the subsequent match of "A" at index 7.

 

If the start pattern or end pattern is not found, or if the end pattern occurs immediately after the start pattern so that there is no substrand between them, snip should return null.

 

If startPattern is the empty string (i.e., "") or null, then this matches the beginning of the strand. Similarly, if endPattern is the empty string or null, then this matches the end of the strand.

 


Finally, modify the findPattern method to allow the pattern to include the
following wildcard characters:

 


* : matches any of the nucleobases
a : matches any base *but* A (i.e., C, G, or T)
c : matches any base but C
t : matches any base but T
g : matches any base but G
